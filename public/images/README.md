# Images Directory

Place your shop images here:

- `logo.png` - Shop logo
- `hero-bg.jpg` - Hero section background image
- `shop-interior.jpg` - Interior photo (optional)
- `team.jpg` - Team photo (optional)

## Image Optimization Tips

1. Use WebP format when possible for better compression
2. Hero images should be at least 1920x1080 for good quality on large screens
3. Compress images before adding (tinypng.com, squoosh.app)
4. Consider using Next.js Image component for automatic optimization
